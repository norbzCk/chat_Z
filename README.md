Chat_Z – A Smart, Self-Learning Python Chatbot

Chat_Z is a simple yet interactive AI chatbot written in Python that learns new responses during conversation, the more info you give it the more it remebers. It stores all interactions in a JSON file, allowing it to remember what it learns even after restarting — just like a mini AI with memory!

Features

->Self-Learning: Learns and remembers new responses on the fly.

->Persistent Memory: Saves learned replies to responses.json.

->Conversational: Responds in both English and Swahili.

->Lightweight: Runs in any Python environment—no external dependencies.

->Simple Structure: Clean, beginner-friendly codebase.

Project Structure Chat_Z/ ├── chatbot.py # Main chatbot script ├── responses.json # Stores learned responses (auto-created) └── README.md
